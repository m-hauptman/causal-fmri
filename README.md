# causal-fmri
Scripts used to analyze fMRI experiments on causal inference.

This folder contains scripts used to analyze data from the following project:
Hauptman, M., & Bedny, M. (2025). Animacy semantic network supports causal inferences about illness. Elife, 13, RP101944.

“Stimuli and procedure” folder: scripts related to selecting and presenting stimuli
“Norming experiment” folder: scripts for analyzing data from online norming experiment
“Analysis” folder: scripts for analyzing data from the fMRI experiment
	> “behavioral” folder contains script for analyzing behavioral responses of participants in the scanner
	> “mvpa” folder contains scripts for performing multivariate analysis, including statistical correction
	> “univariate” folder contains scripts for analyzing univariate data, including PSC extraction

Please contact mhauptm1@jhu.edu with any questions related to these scripts.
